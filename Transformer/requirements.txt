# - Python: 3.11.9
# - pip: 26.1.2
# - 가상환경 위치: 프로젝트 루트의 .venv 폴더
# - PyTorch: CUDA 12.6 빌드(torch 2.12.1+cu126)
#
# 가상환경 생성 및 실행 방법
# 1. 프로젝트 루트에서 가상환경 생성
#    python -m venv .venv
#
# 2. 가상환경 활성화
#    Windows PowerShell:
#    .\.venv\Scripts\Activate.ps1
#
#    macOS/Linux:
#    source .venv/bin/activate
#
# 3. Transformer 패키지 설치
#    pip install -r Transformer/requirements.txt
#
# 4. 가상환경 종료
#    deactivate
